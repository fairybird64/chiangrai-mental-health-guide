# คู่มือระบบดูแลสุขภาพจิตเด็กและวัยรุ่น จ.เชียงราย

เว็บคู่มือสำหรับครูและเครือข่าย พร้อมชุดแบบประเมินคัดกรอง 8 ชุดที่ทำได้จริงในเบราว์เซอร์

## เปิดใช้งาน (GitHub Pages)
1. สร้าง repository ใหม่บน GitHub แล้วอัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ (คงโครงสร้างโฟลเดอร์ `assessments/` ไว้)
2. ไปที่ **Settings → Pages**
3. ที่ **Source** เลือก **Deploy from a branch** → branch `main` → folder `/ (root)` → **Save**
4. รอสักครู่ GitHub จะให้ลิงก์ `https://<username>.github.io/<repo>/` — ส่งลิงก์นี้ให้ทีมได้เลย

## ไฟล์สำคัญ
- `index.html` — หน้าคู่มือหลัก (หน้าแรกของเว็บ)
- `assessments/screening-suite.html` — ชุดแบบประเมิน (PHQ-A, 8Q, ST-5, SNAP-IV, GAST, บคก., PSQI, นิโคติน)
- `support.js` — ตัวรันหน้าเว็บ (ห้ามลบ)
- `คู่มือระบบดูแลสุขภาพจิตนักเรียน.dc.html` — ไฟล์ต้นฉบับสำหรับแก้ไข (แก้ที่นี่แล้วคัดลอกทับ `index.html`)

## หมายเหตุ
- ต้องเชื่อมอินเทอร์เน็ต (ฟอนต์ + ตัวรันหน้าเว็บโหลดจากออนไลน์)
- ลิงก์แบบประเมินเป็นแบบ relative — ต้องเก็บ `index.html` กับโฟลเดอร์ `assessments/` ไว้ด้วยกันเสมอ
